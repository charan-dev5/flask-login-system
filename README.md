# Flask Login System
A secure user authentication system built with Python and Flask.

## Features
- User registration with hashed passwords
- Secure login and logout
- Session management
- Protected pages — redirects to login if not authenticated

## Built with
- Python
- Flask
- Werkzeug (password hashing)
- SQLite3
- HTML

## How to run
1. pip install flask
2. python app.py
3. Open http://127.0.0.1:5000
